# Movies API

[![ESLint](https://github.com/nicholasss/movies-api/actions/workflows/eslint.yml/badge.svg?branch=main)](https://github.com/nicholasss/movies-api/actions/workflows/eslint.yml)

## Tools

To run the server in development mode:

```sh
npm run dev
```

To run ESLint:

```sh
npm run lint
```

To run tests:

```sh
npm run test
```
